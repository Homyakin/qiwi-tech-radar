## 🧑🏼‍💻 Development
| | Command |
|---|---
| Generate | `yarn generate path/to/.cvs output-dir`
| Edit | Specifically have a look at `.eleventy.js` to see if you want to configure
| Test | yarn test
| Preview | `eleventy --serve`
| Debug | `DEBUG=* elevenwty`
